# BloodBank-Management-System
a platform to streamline blood donation, manage donor records and track real time blood inventory to make it faster and efficient to donate and receive blood

Project Description:

The Blood Bank Management System is a web-based application developed using PHP and MySQL. The system is designed to manage blood donors, record donations, maintain blood inventory, and handle hospital blood requests efficiently. The system ensures accurate tracking of available blood units and provides administrative control over all operations.

#Technologies Used:
*Frontend:
HTML
CSS

*Backend:
PHP

*Database:
MySQL (XAMPP)

*Analytics Module:
Python

#Project Modules:

*Donor Management
-Add, edit, delete, and search donor records.

*Donation Management
-Record donations and automatically update blood inventory.

*Inventory Management
-Track blood stock by blood group and monitor availability status.

*Hospital Requests
-Manage hospital blood requests and update status (Pending, Approved, Rejected).

*Authentication
-Admin login system with session-based access control.

#Installation Steps:

Install and start XAMPP (Apache & MySQL).

Create database "bloodbank_db" in phpMyAdmin.

Import the SQL file from sql/database.sql.

Place project folder inside xampp/htdocs/.

Run in browser:
http://localhost/bloodbank/
